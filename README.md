# CMS API

| Method    | End Point             | Description                 |
| --------- |:---------------------:| ---------------------------:|
| GET       | /api/cms              |   Get all cms data          |
| POST      | /api/cms              |   Post a single cms data    |
| PUT       | /api/cms/:cms_id      |    Update a single cms data |
| DELETE    | /api/cms/:cms_id      |    Delete a single cms data |
